第一章 包

什么是包？

  • 包是用来分门别类的管理各种不同程序的，类似于文件夹，建包有利于程序的管理和维护。
  • 建包的语法：

在自己程序中调用其他包下的程序注意事项

  • 如果当前程序中，要调用自己所在包下的其他程序，可以直接调用。（同一个包下的类，互相可以直接调用）
  • 如果当前程序中，要调用其他包下的程序，则必须在当前程序中导包，才可以访问！导包格式：import 包名.类名
  • 如果当前程序中，要调用Java提供的程序，也需要先导包才可以使用，但是Java.lang包下的程序是不需要我们导包的，可以直接使用
  • 如果当前程序中，要调用多个不同包下的程序，而这些程序名正好一样，此刻默认只能导入一个程序，另一个程序必须带包名访问

第二章 String

String的概述

  • String是什么？可以做什么？

    代表字符串，可以用来创建对象封装字符串数据，并对其进行处理

  • String类创建对象封装字符串数据的方式有几种？

    • 方式一：直接使用双引号”… …”。
    • 方法二：new String类，调用构造器初始化字符串对象。

String使用时的注意事项

第一点

  • String对象的内容不可改变，被称为不可变字符串对象。

第二点

  • 只要是以 “…” 方式写出来的字符串对象，会存储到字符串常量池，且相同内容的字符串只存储一份；
  • 但通过new方式创建字符串对象，没new一次都会产生一个新的对象放在堆内存中。

两种方式创建字符串的区别

如果是以双引号创建出来的字符串（例如：String str = “hello”），在内存中只有一份

如果是以new的方法创建出来的字符串(例如：String str1 = new String( ) )，每new一次就有一份

第二章 ArrayList

ArrayList的介绍

  1. 集合是什么，有什么特点？
    • 集合是一种容器，用来存储数据
    • 集合的大小可变
  2. ArrayList是什么？怎么使用？
    • ArrayList是集合中最常用的一种，ArrayList是泛型类，可以约束存储的数据类型。
    • 创建对象，调用无参构造器初始化对象：public ArrayList( )；
    • 调用相应的增删改查的方法

ArrayList的常用方法

构造器 说明
public ArrayList( ) 创建一个空的集合对象
常用方法名 说明
public boolean add (E e) 将指定的元素添加到此集合的末尾
public void add (int index, E element) 在此集合中的指定位置插入指定的元素
public E get (int index) 返回指定元素索引处的元素
public int size( ) 返回集合中的元素的个数
