创建完之后就会在这目录下生成你的文章： blog\myblog\source\_posts，文件格式为.md

编写文章

相信大家对word或者txt已经非常熟悉了，但是对.md后缀的文件可能还比较陌生

接下来将教大家如何在.md文件中创作，并介绍一些我个人常用的方法

标题

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

段落

Markdown的段落没有特殊的格式，直接编写文字就好，段落的换行号是使用两个以上的空格加回车。

字体

  • 用1个星号*或底线_表示斜体
  • 用2个星号*或底线_表示粗体
  • 用3个星号*或底线_表示粗斜体
